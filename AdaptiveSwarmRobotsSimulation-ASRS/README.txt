- In the file demo/demoAutonomousController.tz, change the first line with the full path of the directory in which the needed files 
  for running the simulation are.
  For example "C:\simulations\breve_simulation" (do not write a "\" at the end).

- Run with Breve the file demo/demoAutonomousController.tz

- Press Play button in Breve for running the simulation.

- Once any simulation image appears in Breve, press "b" key for starting the simulation.
